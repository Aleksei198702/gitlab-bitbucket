Откройте терминал или командную строку.

Перейдите в каталог вашего проекта, используя команду cd path/to/your/project, где path/to/your/project - путь к вашему проекту.

Введите следующую команду: npm install express ws

Нажмите Enter.
Команда npm install установит необходимые зависимости (express и ws) и добавит их в файл package.json в раздел "dependencies".

После выполнения этих шагов у вас должна быть установлена необходимая зависимость, и вы сможете запускать ваш сервер с помощью команды npm start
